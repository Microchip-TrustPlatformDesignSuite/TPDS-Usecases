# CryptoAuthLib application examples for Microchip CryptoAuthentication devices

## Contents Summary

| Folder | Description                                                                       |
| ------ | --------------------------------------------------------------------------------- |
| apps   | Example applications for CryptoAuth library components |

## Code Examples

The following applications are provided to demonstrate various usage models using CryptoAuth library components.

| Name                                                                | Description                                                               |
| ------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| [AES Message Encryption](apps/aes_message_encryption/readme.md) | Demonstrates the usage of ECC608 for AES message Encryption|
| [Asymmetric Authentication](apps/asymmetric_authentication/asymm_auth_ta010/readme.md) | Demonstrates the usage of ECC608 and TA010 for Asymmetric Authentication |
| [FOTA with ECC608 TMNGTLS and keySTREAM](apps/keystream_fota/readme.md) | Demonstrates the usage of ECC608-TMNGTLS and keySTREAM for Firmware Over The Air update |
| [Symmetric Authentication Using Diversified Keys](apps/symmetric_authentication/symm_auth_ta010/readme.md) | Demonstrates the usage of ECC608 TFLXTLS and TA010-TFLXAUTH for Derived key based Symmetric Authentication |
| [WPC Authentication with TA010-TFLXWPC](apps/wpc_authentication/wpc_auth_ta010/readme.md) | Demonstrates the usage of TA010-TFLXWPC for WPC Authentication |
